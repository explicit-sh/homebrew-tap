# explicit Homebrew Tap

Homebrew tap for [explicit](https://github.com/explicit-sh/explicit) — real-time Elixir code analysis for Claude Code.

## Install

```bash
brew install explicit-sh/tap/explicit
```

## Usage

```bash
explicit watch              # Start server for current project
explicit status             # Server health
explicit violations --json  # Show violations
explicit stop               # Stop server
```
